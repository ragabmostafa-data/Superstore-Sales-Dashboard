# 📊 Superstore Sales Dashboard

## 📌 Overview

This project presents an interactive Power BI dashboard built to analyze Superstore sales performance.

---

## 🛠 Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX

---

## 📈 KPIs

- Total Sales
- Total Profit
- Total Orders
- Profit Margin

---

## 📊 Dashboard Features

- Sales by Region
- Sales by Category
- Sales Trend
- Top Products
- Customer Segment Analysis

---

## 💡 Key Insights

- Technology category generated the highest sales.
- West region achieved the highest revenue.
- Consumer segment contributed the largest sales.
- Sales increased significantly during the last quarter.
